Pixel Guard is a desktop image steganography application that allows users to
securely hide encrypted messages inside images.

## Features
- Encrypt messages into images
- Decrypt hidden messages
- Password-protected encryption
- 1-channel and 3-channel embedding modes
- Simple, modern GUI

## Installation
Download the installer from the Releases section and run the setup.

## Usage
1. Select an image
2. Enter a message and key
3. Encrypt or decrypt as needed

## Built With
- Python
- Tkinter
- Pillow
- PyInstaller
- Inno Setup
